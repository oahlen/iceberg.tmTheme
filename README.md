# Iceberg Sublime Text

Sublime theme based on [iceberg.vim](https://github.com/cocopon/iceberg.vim) by [cocopon](https://github.com/cocopon)

![Screenshot](https://github.com/oahlen/assets/raw/main/iceberg.tmTheme/screenshot.png)

## Installation [bat](https://github.com/sharkdp/bat) theme

```bash
# Create bat theme directory
mkdir -p "$(bat --config-dir)/themes"

# Download theme file
wget -O "$(bat --config-dir)/themes/iceberg.tmTheme" https://raw.githubusercontent.com/oahlen/iceberg.tmTheme/main/iceberg.tmTheme

# Rebuild bat cache
bat cache --build
```
